# Secure Banking Login Page (Reference-Based UI)

## Overview

This project is a secure, professional banking-style login page built based on a provided visual reference.  
The goal of this assignment is to demonstrate strong frontend fundamentals with a focus on **pixel-accurate UI, clean layout, and basic JavaScript logic**, without using any backend services.

---

## Features

### 1. Login Form
- Username / Member ID input field
- Password input field
- “Sign In” button

### 2. Layout & UI
- Professional, modern financial UI design
- Clean typography and consistent spacing
- Centered login container
- Clear labels and input focus states
- Desktop-first layout (responsive for smaller screens)

### 3. JavaScript Functionality
- Client-side form validation
- Simulated authentication using hardcoded credentials
- Displays appropriate success or error message after submission

---

## Design Requirements

- High attention to spacing, alignment, and proportions
- Colors and typography aligned with modern banking UI standards
- Pixel-accurate implementation based on the provided reference design

---

## Technical Stack

- **HTML5** – Semantic markup
- **CSS3** – Flexbox-based layout
- **JavaScript (Vanilla JS)** – Form handling and validation
- No backend integration
